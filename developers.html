🔌

扩展生态

通过 Geometry Extension Model，你可以编写系统级扩展：文件提供程序、上下文菜单、Shell 插件。

[ExtensionContract("FileProvider")]
public class CloudFS : IExtension {
  async Task<Stream> OpenAsync(string path);
  // 注册后自动出现在文件资源管理器中
}
