系统要求

Windows NEXT Geometry Beta 所需的最低与推荐硬件配置，以及其他必备条件。

💻 硬件规格

组件 最低配置 推荐配置
处理器 1 GHz 双核 x64 2.5 GHz 四核或更高
内存 2 GB RAM 16 GB RAM 或以上
存储 16 GB 可用空间 (SSD 优先) 256 GB NVMe SSD
显卡 DirectX 12 兼容 GPU / WDDM 2.x 支持光线追踪的 GPU (如 NVIDIA RTX 2060 / AMD RX 6600 或更高)
显示屏 1K (1024×768) 或更高 2K 或更高，支持 HDR
固件 使用UEFI引导，支持 Secure Boot （非必须） UEFI 2.5 及以上，支持 Secure Boot
TPM（非必须） TPM 2.0 TPM 2.0 (独立或 fTPM)
* 实际性能取决于设备配置、已安装应用和工作负载。推荐配置可启用全部液态玻璃特效与空间音频。

📋 其他要求

  • Internet 连接： 安装与更新需要连接 Internet。首次设置需 Microsoft 账户。
  • Microsoft 账户： 部分功能（如语义搜索、云同步）需要登录 Microsoft 账户。
  • 显示分辨率： 液态玻璃动态效果在 2K 及以上分辨率表现最佳。
  • 音频： 空间音频引擎需要支持立体声输出的扬声器或耳机。
  • 触控/触控笔： 手势宇宙功能在支持多点触控的设备上体验更佳。
  • 虚拟机： 支持 Hyper-V、VMware Workstation 17+、VirtualBox 7+，需启用嵌套虚拟化。
  • Windows 版本： 可从 Windows 10 1903+ 或 Windows 11 21H2 免费升级安装。
